# Elite Glove Landing Page

A premium, elegant landing page for Elite Glove - a nitrile gloves manufacturer. Features a built-in text editor for easy content customization.

## Features

- **Elegant Dark Theme** - Sophisticated design with gold accents
- **Built-in Text Editor** - Click any text to edit, changes save automatically
- **Responsive Design** - Works on desktop, tablet, and mobile
- **Export Function** - Download clean HTML without editor
- **Local Storage** - Your edits persist between sessions

## How to Use

### Editing Content
1. Click **"Edit Mode"** button in top-right corner
2. Click on any text to edit it
3. Changes auto-save to browser storage
4. Click **"Save"** to manually save

### Exporting
1. Click **"Export"** to download clean HTML
2. The exported file has no editor - ready for production
